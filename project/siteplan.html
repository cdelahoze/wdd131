Site Description

This project arose from the need of the founder of the Talk Time Institute company to teach English with native teachers using technology as an aid.

Talk Time Institute is projected as an online English teaching center for Spanish speakers where students can take English classes with native teachers. The website is fully interactive. Students can practice pronunciation, listening, and dialogue directly on the computer using the Microsoft EDGE browser.

Site Logo

Site Name and Domain

Site Name: Talk Time Institute

The site or domain name (along with a logo) is the primary identifier and is fundamental for branding, visibility, and recognition. Securing an available domain name for purchase is normally important but not mandatory in this class.

Domain: https://talktimeinstitute.com/

Why this name? The name "Talk Time Institute" represents an educational institution focused on conversational English practice, emphasizing speaking time and interaction with native teachers. The domain clearly communicates the purpose of the site to potential students.

Site Purpose

The purpose of the site describes the scope of your site, as well as the services and information that will be provided.

  • Provide online English classes with native teachers for Spanish speakers
  • Enable interactive learning through the Microsoft EDGE browser
  • Offer tools for practicing pronunciation, listening, and dialogue
  • Create an accessible and technology-enhanced learning environment
  • Connect students with qualified native English teachers

Scenarios

Scenarios are questions asked by a site visitor who represents the organization's target audience. The questions will help drive the content of the website as it provides answers to these questions.

Scenario 1:

How can I schedule a trial class with a native English teacher to test the interactive platform?

Scenario 2:

What technology requirements do I need to take classes and practice pronunciation directly from my computer?

Scenario 3:

How does the platform help me improve my listening and speaking skills compared to traditional methods?

WCAG 2.1 Guidelines

  • Charcoal Black on Base Cream: Excellent (AAA). Very high contrast, ideal for long reading.
  • White on Soft Cyan: Good (AA). Readable for large titles, but requires care with small text.
  • Black on Mustard Yellow: Excellent (AAA). Perfect for buttons or highlight labels.
  • Cyan on Yellow: Poor. Do not use these two colors on top of each other for text; use them as adjacent blocks.
